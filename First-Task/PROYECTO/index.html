FIT STYLE

Nuestro proyecto es una aplicación web que ayude al usuario a llevar a cabo una vida saludable, recibiendo varios parametros del usuario para poder obtener los resultados de indice de masa corporal, necesidades y crear una dieta balanceada especifica para ese usuario, además de una rutina de ejercicio específica para el usuario, posiblemente podamos implementar un mapa en donde podamos reocmendar un gimnasio cercano al usuario o mercados o restaurantes nutritivos donde conseguir sus alimentos, además de una alarma para poder comer a las horas indicadas, incluyendo entremeces y otras cosas.
